Using the WebApp Objects

The WebApp application provides the script programmer with access to most of the objects available to the Delphi programmer. The following information was retrieved through these objects :

The name of the machine running the Web server Request.ServerVariables("SERVER_NAME") <%= Request.ServerVariables("SERVER_NAME") %>
The WebApp stub used by this script Request.ScriptName <%= Request.ScriptName%>
The IP address of the machine you are using Request.ServerVariables("REMOTE_HOST") <%= Request.ServerVariables("REMOTE_HOST") %>
The WebApp Application ID Application.AppID <%= Application.AppID%>
The WebApp Session ID Session.SessionID <%= Session.SessionID%>
The Operating System the browser is running on Request.BrowserCaps.Platform <%' Just for fun, let's use the full syntax this time : Response.Write(Request.BrowserCaps.Platform)%>

(There is a lot more where these came from...)
